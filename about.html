NexusLocal

About


This project was made to make games, cloaks, and utilities available for devices with bloatware or any form of restrictions. Some of these can be used offline, such as:

  • JS Console
  • NotePad
  • 3D Editor
  • Blockbench (main functions)
  • Minecraft v1.5.2 (singleplayer)

These tools require Wifi for all their functions to work properly:

  • 1v1.lol (multiplayer)
  • Justfall.lol (multiplayer)
  • about:blank cloak
  • Minecraft v1.5.2 (multiplayer)
  • Blockbench (all functions)

I am constantly adding new games and tools. This project is mainly for computers, but here is a list of things that could possibly work on mobile (not guaranteed):

  • NotePad
  • JS Console
  • about:blank cloak
  • 3D Editor
  • Blockbench
